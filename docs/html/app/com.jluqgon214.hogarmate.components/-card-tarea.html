CardTarea

@Composable
fun CardTarea(task: Tarea, assignedUser: String, onComplete: () -> Unit, onDelete: () -> Unit)

Componente que representa una tarjeta de tarea interactiva.

Este componente permite mostrar información de una tarea, asignarla a un usuario, completarla o eliminarla mediante gestos de deslizamiento.

Parameters

task

Objeto que contiene la información de la tarea.

assignedUser

Nombre del usuario asignado a la tarea.

onComplete

Callback que se ejecuta al marcar la tarea como completada o deshacer su completado.

onDelete

Callback que se ejecuta al eliminar la tarea.
